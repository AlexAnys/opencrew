## 桌面端

**多频道协作总览** — 每个频道对应一个 Agent 岗位，每个频道帖子对应 OpenClaw 的一个 session，切换一目了然

![多频道协作总览](../assets/screenshots/mac-1.jpg)

**待办 - 已完成** — 每条信息一键加入待办/已完成

![待办 - 已完成](../assets/screenshots/mac-2.jpg)

**消息流** — 按时间顺序的消息流，随时跟进最新的对话

![消息流](../assets/screenshots/mac-3.jpg)

**Slack 搜索** — 跨对话或指定频道进行快速搜索

![Slack 搜索](../assets/screenshots/mac-4-search.jpg)
